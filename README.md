Credit Utilization Engine


This project calculates credit utilization for multiple credit cards and classifies risk levels (Low, Medium, High).  
It also provides validation and error handling for incorrect inputs.

---


- Per-card utilization calculation
- Overall utilization calculation
- Risk classification:
  - Low (<30%)
  - Medium (30%–70%)
  - High (>70%)
- Input validation
- Handles duplicate card IDs
- Flask API support

---


- Python
- Flask (for API)

---
